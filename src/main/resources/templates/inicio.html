AirUs

AirUs está destinado al estudio del tráfico aéreo por medio de información geolística de aviones en directo por medio de la Api Rest de OpenSky.

AirUs te permitirá hacer el estudio de distintos aspéctos del tráfico aéreo.

Herramientas

Para el desarrollo de este Trabajo se ha hecho uso de MongoDB como base de datos para almacenar toda la información, Spring como framework para el backend y Bootstrap para frontend.

About Us

AirUs es un trabajo de fin de grado realizado por Rafael Salas Castizo, estudiante de ingeniería informática de software en la Universidad de Sevilla, donde se busca el alamacenamiento, consulta y tratamiento de información geologística de aviones para poder obtener nuevos estudios e información acerca del tráfico aéreo.

Todo esto mediante la supervisión de una tutora, Mª Teresa Gómez López, del departamento de Lenguajes y Sistemas Informáticos de la Universidad de Sevilla.

Api Rest

Para el desarrollo de este trabajo se ha hecho uso de la Api Rest de OpenSky, con la cuál he podido realizar consultas para almacenar la información geologística de aviones en directo y poder implementar diversas funcionalidades que aporten información acerca del tráfico aéreo.
